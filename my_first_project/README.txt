Thank you for coming to the WebApps with cordova.

The commands that will help you get started
(when you download this project you don't need to run these first commands to start
 I'm just putting them in for reference)

installing ionic
npm install -g cordova ionic

ionic start <project name> <template type: blank | tabs | sidemenu

$ cd <project name>
$ ionic platform add ios | android
$ ionic build ios | android
$ ionic run ios | android - installs the app on the device
